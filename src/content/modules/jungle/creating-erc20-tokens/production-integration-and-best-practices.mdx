## Contract Verification on Block Explorer

### Why Contract Verification Matters

**Real-world analogy:** Contract verification is like publishing the source code of an open-source project. It allows anyone to read your code, verify it matches what's deployed, and build trust in your project.

**Benefits of verification:**
- **Transparency:** Users can read your contract's source code
- **Trust:** Proves the deployed bytecode matches your source
- **Integration:** Block explorers can decode function calls and events
- **Debugging:** Easier to understand transaction failures
- **Adoption:** Many users won't interact with unverified contracts

### Verifying Your Contract on Zilliqa Blockscout

**Manual verification process:**

1. **Visit Zilliqa Blockscout:** https://zilliqa.blockscout.com
2. **Search for your contract address**
3. **Click "Verify & Publish"**
4. **Fill in verification details:**
   - **Contract Address:** Your deployed token address
   - **Contract Name:** `ZilliqaAcademyToken`
   - **Compiler Version:** `0.8.28`
   - **Optimization:** `Enabled` with `200` runs
   - **EVM Version:** `shanghai`

5. **Upload contract source:**
   - **Main contract:** Copy your `ZilliqaAcademyToken.sol` content
   - **Constructor parameters:** Encode your constructor parameters

6. **Submit verification**

```typescript
async function main() {
  const contractAddress = "YOUR_CONTRACT_ADDRESS";
  const token = await ethers.getContractAt("ZilliqaAcademyToken", contractAddress);
  
  // Configuration - Replace these with actual values
  const recipient = "RECIPIENT_ADDRESS";
  const amount = ethers.parseEther("100"); // 100 tokens
  
  console.log("ðŸ’¸ Transferring tokens...");
  console.log(`To: ${recipient}`);
  console.log(`Amount: ${ethers.formatEther(amount)} ZAT`);
  
  // Check sender balance first
  const [sender] = await ethers.getSigners();
  const senderBalance = await token.balanceOf(sender.address);
  
  if (senderBalance < amount) {
    console.error("âŒ Insufficient balance for transfer");
    console.log(`Your balance: ${ethers.formatEther(senderBalance)} ZAT`);
    return;
  }
  
  // Execute transfer
  const tx = await token.transfer(recipient, amount);
  console.log(`Transaction hash: ${tx.hash}`);
  console.log("Waiting for confirmation...");
  
  const receipt = await tx.wait();
  console.log(`âœ… Transfer completed!`);
  console.log(`Gas used: ${receipt?.gasUsed}`);
  
  // Show updated balances
  const newSenderBalance = await token.balanceOf(sender.address);
  const recipientBalance = await token.balanceOf(recipient);
  
  console.log(`\nðŸ“Š Updated Balances:`);
  console.log(`Your balance: ${ethers.formatEther(newSenderBalance)} ZAT`);
  console.log(`Recipient balance: ${ethers.formatEther(recipientBalance)} ZAT`);

main()
  .then(() => process.exit(0))
```

### Basic Analytics

**Create `scripts/token-analytics.ts`:**

```typescript
import { ethers } from "hardhat";

async function main() {
  const contractAddress = "YOUR_CONTRACT_ADDRESS";
  const token = await ethers.getContractAt("ZilliqaAcademyToken", contractAddress);
  
  console.log("ðŸ“Š Token Analytics");
  console.log("==================\n");
  
  // Basic token metrics
  const name = await token.name();
  const symbol = await token.symbol();
  const decimals = await token.decimals();
  const totalSupply = await token.totalSupply();
  
  console.log("ðŸ“ˆ Basic Metrics:");
  console.log(`Name: ${name}`);
  console.log(`Symbol: ${symbol}`);
  console.log(`Decimals: ${decimals}`);
  console.log(`Total Supply: ${ethers.formatEther(totalSupply)} ${symbol}\n`);
  
  // Get recent Transfer events
  try {
    const currentBlock = await ethers.provider.getBlockNumber();
    const fromBlock = Math.max(0, currentBlock - 1000); // Last 1000 blocks
    
    const transferEvents = await token.queryFilter(
      token.filters.Transfer(),
      fromBlock,
      currentBlock
    );
    
    console.log("ðŸ“Š Recent Activity (Last 1000 blocks):");
    console.log(`Total Transfers: ${transferEvents.length}`);
    
    if (transferEvents.length > 0) {
      // Calculate unique addresses
      const uniqueAddresses = new Set();
      let totalVolume = 0n;
      
      transferEvents.forEach(event => {
        if (event.args) {
          uniqueAddresses.add(event.args[0]); // from
          uniqueAddresses.add(event.args[1]); // to
          totalVolume += event.args[2]; // value
        }
      });
      
      console.log(`Unique Active Addresses: ${uniqueAddresses.size}`);
      console.log(`Total Transfer Volume: ${ethers.formatEther(totalVolume)} ${symbol}`);
      
      // Most recent transfer
      const latestTransfer = transferEvents[transferEvents.length - 1];
      if (latestTransfer.args) {
        console.log(`\nMost Recent Transfer:`);
        console.log(`  Amount: ${ethers.formatEther(latestTransfer.args[2])} ${symbol}`);
        console.log(`  Block: ${latestTransfer.blockNumber}`);
      }
    }
    
  } catch (error) {
    console.log("Note: Historical event analysis requires archive node access");
  }
  
  console.log(`\nðŸ”— View on Block Explorer:`);
  console.log(`https://zilliqa.blockscout.com/address/${contractAddress}`);
}

main()
  .then(() => process.exit(0))
  .catch((error) => {
    console.error(error);
    process.exit(1);
  });
```
