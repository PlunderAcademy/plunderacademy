## Deploying Your Simple Token

### Quick Deployment with Hardhat Ignition

Let's deploy your simple token to Zilliqa testnet and start interacting with it!

**Create deployment module `ignition/modules/MyFirstToken.ts`:**

```bash
rm ignition/modules/Counter.ts
```

```typescript
import { buildModule } from "@nomicfoundation/hardhat-ignition/modules";

const MyFirstTokenModule = buildModule("MyFirstTokenModule", (m) => {
  // Get the claimant address parameter
  const claimant = m.getParameter("claimant");
  
  // Deploy the token contract with claimant address
  const token = m.contract("MyFirstToken", [claimant]);
  
  return { token };
});

export default MyFirstTokenModule;
```

**Important:** The deployment module now requires a `claimant` parameter - this should be your connected MetaMask wallet address!

### Deploy to Zilliqa Testnet

**Make sure your environment is ready:**

1. **Check your `.env` file has your private key:**
   ```
   PRIVATE_KEY=your_metamask_private_key_here
   ```

2. **Ensure you have testnet ZIL** in your MetaMask wallet

3. **Deploy your token (replace YOUR_WALLET_ADDRESS with your MetaMask address):**
   ```bash
   npx hardhat ignition deploy ignition/modules/MyFirstToken.ts --network zilliqaTestnet --parameters '{"MyFirstTokenModule": {"claimant": "YOUR_WALLET_ADDRESS"}}'
   ```

**Example with actual address:**
   ```bash
   npx hardhat ignition deploy ignition/modules/MyFirstToken.ts --network zilliqaTestnet --parameters '{"MyFirstTokenModule": {"claimant": "0x742d35Cc6565C42d26A7b4b4c6d3A5c5c5d4e4F4"}}'
   ```

**Expected output:**

## Interacting with Your Token

### Using Hardhat Console

**Connect to your deployed token:**

```bash
npx hardhat console --network zilliqaTestnet
```

**In the console:**

```javascript
// Connect to your token
const token = await ethers.getContractAt("MyFirstToken", "YOUR_CONTRACT_ADDRESS");

// Check basic info
await token.name()        // "My First Token"
await token.symbol()      // "MFT"
await token.decimals()    // 18n

// Check total supply
const totalSupply = await token.totalSupply();
console.log(`Total Supply: ${ethers.formatEther(totalSupply)} MFT`);

// Check your balance
const [deployer] = await ethers.getSigners();
const balance = await token.balanceOf(deployer.address);
console.log(`Your Balance: ${ethers.formatEther(balance)} MFT`);
```

### Transfer Tokens

**Send tokens to another address:**

```javascript
// In Hardhat console
const recipientAddress = "0x742d35Cc65C42d26A7b4b4c6d3A5c5c5d4e4F4"; // Replace with actual address
const transferAmount = ethers.parseEther("1000"); // 1000 MFT

console.log("Transferring 1000 MFT...");
const tx = await token.transfer(recipientAddress, transferAmount);
console.log(`Transaction hash: ${tx.hash}`);

// Wait for confirmation
await tx.wait();
console.log("Transfer completed!");

// Check new balances
const yourBalance = await token.balanceOf(deployer.address);
const recipientBalance = await token.balanceOf(recipientAddress);

console.log(`Your new balance: ${ethers.formatEther(yourBalance)} MFT`);
console.log(`Recipient balance: ${ethers.formatEther(recipientBalance)} MFT`);
```

## Verifying Your Token on Block Explorer

### Step-by-Step Verification

1. **Visit Zilliqa Blockscout:** https://zilliqa.blockscout.com

2. **Search for your contract:**
   - Paste your contract address in the search bar
   - Click on your contract address in the results

3. **Navigate to verification:**
   - Click on the "Contract" tab
   - Click "Verify & Publish"

4. **Fill in the details:**
   - **Contract Address:** (Auto-filled)
   - **Contract Name:** `MyFirstToken`
   - **Compiler Version:** `0.8.28`
   - **Optimization:** `Yes`
   - **Optimization Runs:** `200`
   - **EVM Version:** `shanghai`

5. **Upload contract source:**
   - Copy the contents of your `contracts/MyFirstToken.sol`
   - Paste it in the "Contract Source Code" field

6. **Submit verification:**
   - Click "Verify and Publish"
   - Wait for verification (usually takes 1-2 minutes)

**Success!** Your contract should now show a green checkmark and "Contract Source Code Verified".

## Creating Transfer Scripts

### Basic Transfer Script

**Create `scripts/transfer-tokens.ts`:**

```typescript
import { ethers } from "hardhat";

async function main() {
  const contractAddress = "YOUR_CONTRACT_ADDRESS"; // Replace with your address
  const token = await ethers.getContractAt("MyFirstToken", contractAddress);
  
  // Configuration
  const recipientAddress = "RECIPIENT_ADDRESS"; // Replace with recipient
  const amount = ethers.parseEther("100"); // 100 MFT
  
  const [sender] = await ethers.getSigners();
  
  console.log("=== Token Transfer ===");
  console.log(`From: ${sender.address}`);
  console.log(`To: ${recipientAddress}`);
  console.log(`Amount: ${ethers.formatEther(amount)} MFT`);
  
  // Check sender balance
  const senderBalance = await token.balanceOf(sender.address);
  console.log(`Sender balance: ${ethers.formatEther(senderBalance)} MFT`);
  
  if (senderBalance < amount) {
    console.error("‚ùå Insufficient balance!");
    return;
  }
  
  // Execute transfer
  console.log("\n‚è≥ Sending transaction...");
  const tx = await token.transfer(recipientAddress, amount);
  console.log(`Transaction hash: ${tx.hash}`);
  
  // Wait for confirmation
  console.log("‚è≥ Waiting for confirmation...");
  const receipt = await tx.wait();
  console.log(`‚úÖ Transfer confirmed in block ${receipt?.blockNumber}`);
  console.log(`Gas used: ${receipt?.gasUsed}`);
  
  // Show updated balances
  const newSenderBalance = await token.balanceOf(sender.address);
  const recipientBalance = await token.balanceOf(recipientAddress);
  
  console.log("\nüìä Updated Balances:");
  console.log(`Sender: ${ethers.formatEther(newSenderBalance)} MFT`);
  console.log(`Recipient: ${ethers.formatEther(recipientBalance)} MFT`);
  
  console.log(`\nüîç View on Explorer:`);
  console.log(`https://zilliqa.blockscout.com/tx/${tx.hash}`);
}

main()
  .then(() => process.exit(0))
  .catch((error) => {
    console.error(error);
    process.exit(1);
  });
```

**Run the transfer:**

```bash
npx hardhat run scripts/transfer-tokens.ts --network zilliqaTestnet
```
