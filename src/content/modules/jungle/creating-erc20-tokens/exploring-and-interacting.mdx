## Interacting Your Simple Token

Let's start interacting with your simple token on Zilliqa testnet!

## Adding Your Token to MetaMask

### Step-by-Step Token Addition

**Manual addition process:**

**Open MetaMask** and ensure you're on Zilliqa EVM Testnet

**Go to Assets tab** and click "Import tokens"

**Select "Custom Token"**

**Enter token details:**
   - **Token Contract Address:** Your deployed contract address
   - **Token Symbol:** (should auto-populate)
   - **Token Decimals:** (should auto-populate)

**Click "Add Custom Token"** then "Import Tokens"

**Your token should now appear in MetaMask!** You should see your balance of 1,000,000 MFT tokens.

## Creating Transfer Scripts

### Basic Transfer Script

**Create `scripts/transfer-tokens.ts`:**

```typescript
import { network } from "hardhat";
import { formatEther, parseEther } from "viem";

async function main() {
  // Configuration - replace with your values
  const contractAddress = process.env.CONTRACT_ADDRESS;
  const recipientAddress = process.env.RECIPIENT_ADDRESS;
  
  if (!contractAddress || !recipientAddress) {
    console.error("‚ùå Please provide CONTRACT_ADDRESS and RECIPIENT_ADDRESS environment variables");
    console.log("Usage: CONTRACT_ADDRESS=0x... RECIPIENT_ADDRESS=0x... npm run transfer");
    process.exit(1);
  }

  const networkName = process.env.HARDHAT_NETWORK || "localhost";
  console.log(`Transferring tokens on ${networkName}...`);
  
  try {
    const { viem } = await network.connect({
      network: networkName,
    });

    const publicClient = await viem.getPublicClient();
    const [walletClient] = await viem.getWalletClients();
    
    // Get the contract instance
    const token = await viem.getContractAt("MyFirstToken", contractAddress as `0x${string}`);
    
    // Transfer amount (100 MFT)
    const amount = parseEther("100");
    
    console.log("=== Token Transfer ===");
    console.log(`From: ${walletClient.account.address}`);
    console.log(`To: ${recipientAddress}`);
    console.log(`Amount: ${formatEther(amount)} MFT`);
    
    // Check sender balance
    const senderBalance = await token.read.balanceOf([walletClient.account.address]);
    console.log(`Sender balance: ${formatEther(senderBalance)} MFT`);
    
    if (senderBalance < amount) {
      console.error("‚ùå Insufficient balance!");
      return;
    }
    
    // Execute transfer
    console.log("\n‚è≥ Sending transaction...");
    const transferTx = await token.write.transfer([recipientAddress as `0x${string}`, amount]);
    console.log(`Transaction hash: ${transferTx}`);
    
    // Wait for confirmation
    console.log("‚è≥ Waiting for confirmation...");
    const receipt = await publicClient.waitForTransactionReceipt({ 
      hash: transferTx 
    });
    console.log(`‚úÖ Transfer confirmed in block ${receipt.blockNumber}`);
    console.log(`Gas used: ${receipt.gasUsed}`);
    
    // Show updated balances
    const newSenderBalance = await token.read.balanceOf([walletClient.account.address]);
    const recipientBalance = await token.read.balanceOf([recipientAddress as `0x${string}`]);
    
    console.log("\nüìä Updated Balances:");
    console.log(`Sender: ${formatEther(newSenderBalance)} MFT`);
    console.log(`Recipient: ${formatEther(recipientBalance)} MFT`);
    
    console.log(`\nüîç View on Explorer:`);
    console.log(`https://testnet.zilliqa.blockscout.com/tx/${transferTx}`);
    
  } catch (error) {
    console.error("‚ùå Transfer failed:", error);
    process.exit(1);
  }
}

main().catch(console.error);
```

## Token Analytics

### Basic Token Stats

**Create `scripts/token-stats.ts`:**

```typescript
import { network } from "hardhat";
import { formatEther } from "viem";

async function main() {
  const contractAddress = process.env.CONTRACT_ADDRESS;
  const networkName = process.env.HARDHAT_NETWORK || "localhost";
  
  if (!contractAddress) {
    console.error("‚ùå Please provide CONTRACT_ADDRESS environment variable");
    console.log("Usage: CONTRACT_ADDRESS=0x... npm run stats");
    process.exit(1);
  }

  console.log(`Analyzing token on ${networkName}...`);
  
  try {
    const { viem } = await network.connect({
      network: networkName,
    });

    const [walletClient] = await viem.getWalletClients();
    const token = await viem.getContractAt("MyFirstToken", contractAddress as `0x${string}`);
    
    console.log("üìä My First Token Analytics");
    console.log("============================");
    
    // Basic token info
    console.log("\nü™ô Token Information:");
    const name = await token.read.name();
    const symbol = await token.read.symbol();
    const decimals = await token.read.decimals();
    const totalSupply = await token.read.totalSupply();
    const claimant = await token.read.claimant();
    
    console.log(`Name: ${name}`);
    console.log(`Symbol: ${symbol}`);
    console.log(`Decimals: ${decimals.toString()}`);
    console.log(`Total Supply: ${formatEther(totalSupply)} MFT`);
    console.log(`Claimant: ${claimant}`);
    
    // Contract info
    console.log("\nüìã Contract Information:");
    console.log(`Address: ${contractAddress}`);
    console.log(`Explorer: https://testnet.zilliqa.blockscout.com/address/${contractAddress}`);
    
    // Current user info
    console.log("\nüë§ Current User Information:");
    console.log(`Address: ${walletClient.account.address}`);
    const userBalance = await token.read.balanceOf([walletClient.account.address]);
    console.log(`Balance: ${formatEther(userBalance)} MFT`);
    console.log(`Percentage: ${((Number(userBalance) / Number(totalSupply)) * 100).toFixed(2)}%`);
    
    // Claimant info
    console.log("\nüèÜ Claimant Information:");
    const claimantBalance = await token.read.balanceOf([claimant]);
    console.log(`Balance: ${formatEther(claimantBalance)} MFT`);
    console.log(`Percentage: ${((Number(claimantBalance) / Number(totalSupply)) * 100).toFixed(2)}%`);
    
    // Distribution summary
    console.log("\nüìä Supply Distribution:");
    const userBalanceFormatted = Number(formatEther(userBalance));
    const claimantBalanceFormatted = Number(formatEther(claimantBalance));
    const totalSupplyFormatted = Number(formatEther(totalSupply));
    const otherBalance = totalSupplyFormatted - userBalanceFormatted - claimantBalanceFormatted;
    
    console.log(`Current User: ${userBalanceFormatted.toLocaleString()} MFT`);
    console.log(`Claimant: ${claimantBalanceFormatted.toLocaleString()} MFT`);
    console.log(`Others: ${otherBalance.toLocaleString()} MFT`);
    console.log(`Total: ${totalSupplyFormatted.toLocaleString()} MFT`);
    
  } catch (error) {
    console.error("‚ùå Analytics failed:", error);
    process.exit(1);
  }
}

main().catch(console.error);
```

**Run the analytics:**

```bash
# Replace with your contract address
CONTRACT_ADDRESS=0x564498e20b1d239d038d6c214f826a44527d6c77 npm run stats:testnet
```

## Exploring the Block Explorer

### Understanding Your Token's Page

**Visit your token on Blockscout:**
`https://testnet.zilliqa.blockscout.com/address/YOUR_CONTRACT_ADDRESS`

**Key sections to explore:**

**Overview Tab:**
   - Contract balance and transaction count
   - Contract creation details
   - Recent transactions and transfers

**Transactions Tab:**
   - All interactions with your token
   - Transfer events and gas usage
   - Internal transactions

**Contract Tab:**
   - Verified source code (after running hardhat verify)
   - Read/Write contract functions
   - Contract ABI and bytecode

**Events Tab:**
   - Transfer events showing token movements
   - Event details and transaction hashes
   - Event filters and search

### Reading Contract Functions

**After verification, try these read functions on the explorer:**

1. **name()** ‚Üí "My First Token"
2. **symbol()** ‚Üí "MFT"
3. **decimals()** ‚Üí 18
4. **totalSupply()** ‚Üí 1000000000000000000000000 (1M tokens in wei)
5. **balanceOf(address)** ‚Üí Check any address balance
